# 🎯 Number Guessing Game

A simple and interactive Number Guessing Game built using **HTML**, **CSS**, and **JavaScript**. The player has to guess a randomly generated number between **1 and 10**. The game provides hints whether the guess is too high or too low and tracks the number of attempts.

## ✨ Features

- 🎲 Random secret number generation
- 🔢 Guess numbers between 1 and 10
- 📈 Displays "Too High" or "Too Low" hints
- 🎉 Shows success message when guessed correctly
- 🔄 New Game button to restart the game
- 📊 Tracks the number of attempts
- 🎨 Responsive and modern UI

## 🚀 How to Run

1. Clone this repository

```bash
git clone https://github.com/your-username/Number-Guessing-Game.git
```

2. Open the project folder.

3. Double-click **index.html** or open it in your browser.

---

## 🎮 How to Play

1. Enter a number between **1 and 10**.
2. Click the **Guess** button.
3. The game will tell you if your guess is:
   - 📉 Too Low
   - 📈 Too High
   - 🎉 Correct
4. The number of attempts will be displayed.
5. Click **New Game** to start again with a new random number.
